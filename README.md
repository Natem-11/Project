## Project Description
Project Rubix is a communication‑focused web application designed to help users connect
safely and engage through one‑on‑one messaging, group chats, and daily app-wide driven
content. The platform focuses on user safety, age restrictions, and positive communication,
ensuring that everyone using the app can enjoy a fun, secure, and welcoming environment.
A standout feature of Project Rubix is the “Video of the Day”, a daily interactive video that
encourages users to react, share opinions, and meet new people based on their responses.
This feature will create for users a way for them to expand there verizons and meet all types of
new people that can be beneficial to their well-being.
Project Rubix will also offer a premium subscription tier, Rubix+, which unlocks additional
customization and features like:
